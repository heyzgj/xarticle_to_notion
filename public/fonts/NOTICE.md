Lope self-hosts these fonts (MV3 forbids CDN font loading). All are free to bundle/redistribute.

| Font | Files | License | Source |
|------|-------|---------|--------|
| General Sans | `GeneralSans-{500,600,700}.woff2` | ITF Free Font License | https://www.fontshare.com/fonts/general-sans |
| Geist | `Geist.woff2` (variable) | SIL Open Font License 1.1 | https://github.com/vercel/geist-font |
| Geist Mono | `GeistMono.woff2` (variable) | SIL Open Font License 1.1 | https://github.com/vercel/geist-font |

- Geist / Geist Mono OFL text: see `LICENSE-Geist-OFL.txt` (ships alongside the fonts, per OFL §clause requiring the license to accompany the font).
- General Sans is provided under the Indian Type Foundry Free Font License (free for commercial + personal use, redistribution permitted). Full terms: https://www.fontshare.com/licenses/itf-ffl

Note: the OFL prohibits selling the fonts on their own and reserves no reserved font names here. Bundling inside an application is expressly permitted.
